Graphes¶

Le module matplotlib permet d’afficher des graphes. Pour utiliser ses fonctions nous devons importer ces modules.

import matplotlib.pyplot as plt
import numpy as np

Créons un vecteur avec des données que nous pouvons afficher.

y = [2, 4, 3, 6, 3]

Un simple plot est obtenu avec ceci.

plt.plot(y)

[<matplotlib.lines.Line2D at 0x7fb7f976ffa0>]

L’objet retourné est de type Line2D. Dorénavant nous terminons la dernière ligne avec un point-virgule ; pour supprimer cet output.

Attention pour afficher un graphe, vous devez ajouter plt.show() dans votre programme Python. Dans ce document-ci cette commande est inclut de façon implicit.

Titres et légendes¶

Nous pouvons ajouter un titre pour le graphe ainsque que pour les deux axes x et y.

plt.title('Evolution')
plt.xlabel('temps')
plt.ylabel('montant')
plt.plot(y);

plt.plot(y, 'or', y);

Coordonnées dans x¶

Si nous affichons un graphe à partir d’un seul vecteur, les valeurs x sont sont des entiers commençant à 0. Nous pouvons définir notre propre plage x, par exemple avec un vecteur x allant de -2 à +2.

Le style de la ligne est indiqué avec un texte:

  • 'g^:' green, triangle, pointillé

  • 'or' discs, red

x = [-2, -1, 0, 1, 2]
plt.plot(x, y,'g^:');

Propriétés de ligne¶

Avec des arguments à options nous pouvons changer les propriétés d’une ligne.

plt.plot(x, y, linewidth=2, color='r', linestyle='--', marker='o');

Histogramme¶

Un diagramme à barres est aussi appelé histogramme. Il représente des des informations sous forme de barres.

plt.bar(x, y)

<BarContainer object of 5 artists>

Sous-graphes¶

Multiples graphes peuvent être combinés. La commande subplot(xyz) designe le z-ième graphe de dimensions \(x \times y\)

plt.subplot(121)
plt.bar(x, y)
plt.subplot(122)
plt.plot(y, 'or-');

Nous pouvons aussi sous-diviser en \(2 \times 2\) sous-graphes.

plt.subplot(221)
plt.bar(x, y)
plt.subplot(222)
plt.plot(y, 'ro-');
plt.subplot(223)
plt.plot(x, y)
plt.subplot(224)
plt.plot(y, 'og');

Créer des données aléatoires.

Multiples lignes¶

Nous pouvons placer multiples lignes sur un plot.

t = np.arange(0., 2., 0.2)
plt.plot(t, t, 'r--', t, t**2, 'bs:', t, t**3, 'g^:');

Courbes sinusoidale¶

t = np.arange(0, 2, 0.01)
s = 1 + np.sin(2 * np.pi * t)
plt.title('signal sinusoidale')
plt.ylabel('voltage (mV)')
plt.xlabel('temps (s)')
plt.grid()
plt.plot(t, s);

Bar plot¶

plt.bar(['a', 'b', 'c'], (2, 3, 5));

numeric x-axis

plt.bar((1, 2, 5), (2, 3, 1));

Fréquence de lettres¶

Un dictionnaire est une structure qui permet de compter facilement les lettres dans une phrase.

s = 'Introduction to python programming'
d = {}
for c in s:
    d[c] = d.get(c, 0) + 1

Ceci nous permet de dessiner un histogramme.

plt.bar(d.keys(), d.values());

Histogramme superposé¶

labels = ['G1', 'G2', 'G3', 'G4', 'G5']
men_means = [20, 35, 30, 35, 27]
women_means = [25, 32, 34, 20, 25]
men_std = [2, 3, 4, 1, 2]
women_std = [3, 5, 2, 3, 3]
width = 0.35       # the width of the bars: can also be len(x) sequence

fig, ax = plt.subplots()

ax.bar(labels, men_means, width, yerr=men_std, label='Men')
ax.bar(labels, women_means, width, yerr=women_std,
        bottom=men_means,
        label='Women')

ax.set_ylabel('Scores')
ax.set_title('Scores by group and gender')
ax.legend()

plt.show()

Figures de Lissajous¶

Les figures des Lissajous sont des combinaisons de courbes sinus et cosinus.

t = np.arange(0, 10, 0.1)
plt.grid()
plt.plot(t, np.sin(3*t), 'r');

plt.grid()
plt.plot(np.sin(t), np.cos(t), 'r');

plt.grid()
plt.plot(np.sin(t), np.cos(3*t), 'r');

plt.grid()
plt.plot(np.sin(2*t), np.cos(3*t), 'r');

Echelle logarithmiques¶

plt.xscale('log')

plt.xscale('log')
plt.yscale('log')

Coordonnées polaire¶

r = np.arange(0, 2, 0.01)
theta = 2 * np.pi * r

plt.subplots(subplot_kw={'projection': 'polar'})
plt.plot(theta, r)
plt.gca().set_rticks([0.5, 1, 1.5, 2])

plt.title("A line plot on a polar axis");
